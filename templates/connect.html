Connect API

Our API allows you to query your data in multiple ways, making it easy to pull your data into your own analysis program or share with colleagues. All of the output are in JSON format.

Projects
To get the list of projects and attributes for an account:
http://{account}.digitalbiome.io/api/v1/projects

Sites
To get all sites:
http://{account}.digitalbiome.io/api/v1/sites
To get sites for a specific project:
http://{account}.digitalbiome.io/api/v1/sites/{project}

Data
To get data for a project, with optional filter parameters (sites, columns, start, end):
http://{account}.digitalbiome.io/api/v1/data/{project}?sites={site1,site2,...}&columns={col1,col2,...}&start={YYYY-mm-dd}&end={YYYY-mm-dd}

Try out some examples:

http://demo.digitalbiome.io/api/v1/projects

http://demo.digitalbiome.io/api/v1/data/demo-project?sites=site1
